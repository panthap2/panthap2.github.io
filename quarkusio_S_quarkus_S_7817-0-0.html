Describe the bug
I have a gradle multi module setup. One module is used as a library, the other is the quarkus application using this library. The library use and includes the KubernetesClient. When I also add the smallrye-health extension to the library, the application can no longer inject the KubernetesClient. The same setup works with Quarkus 1.2.1.Final.

Expected behavior
KubernetesClient should be injected.

Actual behavior
