hssm commented Jan 16, 2020

The ETA function is identical in both versions so you won't find a difference if you switch schedulers.

The function looks at the last 7 days in the revlog history and uses an average of rep time for certain reps. It's possible you have an outlier or two dragging the average up and skewing the estimate. It should clear up in a week on its own if long reps aren't a regular thing for you.

If you are keen, you can use the Advanced Browser add-on on the desktop and add the Previous Duration column to find the slowest rep.
