Comment #18

jeremylong commented Mar 25, 2020

In some cases the environment path on your command line is different than the path within the JVM. In this case you will see the error "...dotnet executable could not be found on the path..." The error message gives you the exact steps you need to use to resolve the issue. See the CLI documentation - you just need to add --dotnet c:/path/to/dotnet.
