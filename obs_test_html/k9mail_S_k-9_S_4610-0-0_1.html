SebiderSushi commented Mar 15, 2020

Expected behavior

When "Use app theme" is selected for either Message view or Composer theme, it should apply a theme according to the selection of "Theme".

Actual behavior

On my device, K-9 Mail uses the Android 8+ " System theme" if "Use app theme" is selected for either Message view or Composer theme.

Steps to reproduce

  1. Set preferences according to "Theme=Light", "Message view theme=Use app theme"
  2. Open a message -> Light themed
  3. Go to System settings > Display > Style > [Switch to dark theme] ( Steps apply on LineageOS )
  4. Reopen the message -> Dark themed

Environment

K-9 Mail version: 5.705 (27005) from F-Droid

Android version: LineageOS 15.1 based on AOSP 8.1

Account type (IMAP, POP3, WebDAV/Exchange): IMAP
